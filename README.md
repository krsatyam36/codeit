# ⚡ Codeit

**Research to Code Engine** Transform academic research papers, mathematical text, and algorithms directly into executable PyTorch Jupyter Notebooks—completely offline using local LLMs.

---

## 🚀 Features

* **Privacy-First Local Execution:** Powered by local Ollama nodes. No API keys, no cloud data leaks, no subscription costs.
* **Dual Input Modes:** Upload local PDF documents (up to 25MB) or paste a direct arXiv URL to scrape papers dynamically.
* **Modern Glassmorphic UI:** A sleek, dark-themed, highly interactive dashboard built with Tailwind CSS.
* **Real-Time Terminal Streaming:** Watch the model "think" with a developer-centric terminal stream that outputs the generation process live to your browser via Server-Sent Events (SSE).
* **Resilient Engine:** Features a custom fallback heuristic parser that seamlessly handles unescaped JSON strings and broken markdown fences generated by LLMs.
* **Instant Compilation:** Automatically assembles valid `.ipynb` files with structured markdown headers and Python code cells, triggering an automatic download when complete.

## 🛠️ Tech Stack

* **Backend:** Python, FastAPI, Uvicorn
* **Document Processing:** PyMuPDF (`fitz`), `httpx` (for arXiv scraping)
* **AI Orchestration:** Ollama API (`qwen2.5-coder:14b-instruct`)
* **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript
* **CLI Engine:** Click
* **Notebook Generation:** `nbformat`

## 📦 Prerequisites

Before running Codeit, ensure you have the following installed on your local machine:
1. **Python 3.10+**
2. **Ollama** (running locally)
3. The specific Llama model pulled to your machine:
   ```bash
   ollama run qwen2.5-coder:14b-instruct

⚙️ Installation

    Clone the repository:
    Bash

    git clone [https://github.com/krsatyam36/codeit.git](https://github.com/krsatyam36/codeit.git)
    cd codeit

    Set up a virtual environment (Recommended):
    Bash

    python3 -m venv venv
    source venv/bin/activate

    Install the package:
    Bash

    pip install -e .

💻 Usage

Start the local compilation server using the built-in CLI:
Bash

codeit serve

The system will automatically spin up the Uvicorn server and open your default web browser to http://localhost:8080.
